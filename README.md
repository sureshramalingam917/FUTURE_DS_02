# Customer Churn Analysis Dashboard

## 📊 Project Overview
This Power BI project analyzes customer churn behavior and helps understand why customers leave and how to improve retention.

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📌 Key Insights
- Month-to-month customers have highest churn
- Low tenure customers are most likely to leave
- Long-term contracts reduce churn significantly

## 💡 Recommendations
- Promote yearly contracts
- Improve onboarding experience
- Give retention offers for new users
